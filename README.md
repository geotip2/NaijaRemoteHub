# NaijaRemoteHub
NaijaRemoteHub 🇳🇬

A membership platform for Nigerians to access remote jobs, templates, and community.

## Tech Stack
- Python
- Streamlit 
- FastAPI for webhooks
- SQLite Database
- Flutterwave for ₦ Payments

## How to Run Locally
1. Install requirements: `pip install -r requirements.txt`
2. Set environment variables: `FLW_SECRET_KEY` and `APP_URL`
3. Run: `streamlit run main.py`

## Features
- User Auth with bcrypt
- Flutterwave Bank Transfer in NGN
- Auto payment confirmation via Webhook
- Dashboard with Job Leads, Templates, Community
- 15% Referral System
- Admin Panel

## Deploy
This app is deployed on Render.com
